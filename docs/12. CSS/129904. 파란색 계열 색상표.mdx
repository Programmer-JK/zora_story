## 파란색 색상 사용 가이드

이 색상표는 웹 디자인에서 파란색 계열을 효과적으로 활용하고 싶은 개발자와 디자이너를 위해 제작되었습니다. 각 색상은 다음과 같은 용도로 활용하실 수 있습니다:

- Light 계열 색상들은 배경이나 강조하고 싶지 않은 요소에 적합합니다
- Medium 계열 색상들은 일반적인 강조나 버튼 등의 인터랙티브 요소에 좋습니다
- Dark 계열 색상들은 텍스트나 중요한 강조 요소에 사용하기 적합합니다
